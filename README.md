# House Price Prediction using Gradient Boosting

This project implements a machine learning model to predict house prices using **Gradient Boosting Regression**. It demonstrates the complete workflow of a regression problem, from data generation to model evaluation.

### Key Features
* **Synthetic Dataset:** Generates a realistic dataset with 5 features (Size, Bedrooms, Age, etc.) using Scikit-Learn.
* **Gradient Boosting:** Utilizes `GradientBoostingRegressor` for high-performance prediction.
* **Evaluation Metrics:** Calculates **MAE** (Mean Absolute Error) and **RMSE** (Root Mean Squared Error) to quantify accuracy.
* **Visualization:** Includes a **Residual Plot** using Matplotlib to analyze prediction errors and check for model bias.

### Technologies Used
* Python
* Scikit-Learn
* Pandas & NumPy
* Matplotlib
